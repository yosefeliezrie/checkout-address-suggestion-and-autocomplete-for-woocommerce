Allows your customers to Autocomplete billing and shipping address in checkout page with google places API.

== Description ==

Checkout Address suggestion and Autocomplete for woocommerce simplifies the checkout process helping your customers to enter their address with the Google Places suggest API. It will save time and prevent typing errors.

Checkout Address suggestion and Autocomplete help to Decreases cart abandonment by checking out quickly.

Checkout Address Suggestion and Autocomplete is an extension that allows your customers to Autocomplete billing and shipping address in checkout page with google places API

= Features: =

* Supports latest woocommerce
* Save your customer time and prevent typing errors
* Checkout Address Sugestion and autocomplete
* Helping your customers to enter their address with the Google Places suggest API.
* Decreases cart abandonment by checking out quickly


== Installation ==

To install Checkout Address suggestion and Autocomplete for woocommerce, follow these steps:

1.Download and unzip the plugin

2.Upload the entire woo-checkout-address/ directory to the /wp-content/plugins/ directory

3.Activate the plugin through the Plugins menu in WordPress
